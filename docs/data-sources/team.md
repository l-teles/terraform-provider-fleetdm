# fleetdm_team (Data Source)

> **Deprecated:** `fleetdm_team` is deprecated and will be removed in a future version. Use [`fleetdm_fleet`](../data-sources/fleet.md) instead. Requires Fleet 4.82.0+.

Retrieves information about a FleetDM fleet.

## Example Usage

```terraform
# Get a specific team by ID
data "fleetdm_team" "workstations" {
  id = 1
}

# Use team data in other resources
output "team_name" {
  value = data.fleetdm_team.workstations.name
}

output "team_host_count" {
  value = data.fleetdm_team.workstations.host_count
}
```
