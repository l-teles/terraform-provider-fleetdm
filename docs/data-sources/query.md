# fleetdm_query (Data Source)

> **Deprecated:** `fleetdm_query` is deprecated and will be removed in a future version. Use [`fleetdm_report`](../data-sources/report.md) instead. Requires Fleet 4.82.0+.

Use this data source to retrieve information about a specific FleetDM query.

## Example Usage

```terraform
# Get a specific query by ID
data "fleetdm_query" "os_version" {
  id = 1
}

# Use query data
output "query_name" {
  value = data.fleetdm_query.os_version.name
}

output "query_sql" {
  value = data.fleetdm_query.os_version.query
}

output "query_interval" {
  value = data.fleetdm_query.os_version.interval
}
```
