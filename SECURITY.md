# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this Terraform provider, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Instead, use [GitHub's private vulnerability reporting](https://github.com/l-teles/terraform-provider-fleetdm/security/advisories/new).
3. Include as much detail as possible: steps to reproduce, affected versions, and potential impact.

You should receive an initial response within 72 hours. Once the issue is confirmed, a fix will be developed and released as soon as possible, with credit given to the reporter (unless anonymity is preferred).
